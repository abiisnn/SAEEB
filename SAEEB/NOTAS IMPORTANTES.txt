Al cargar la BD .sql
1.- Abrir MySQL
2.- Simplemente escribir "source" y arrastrar y pegar el archivo .sql
3.- Y ya(: la bd se llama saeeb (automaticamente crea la bd y la selecciona con use)

Codigo de IdUsuarios:
- Inicia con 2 -> Alumno
- Inicia con 3 -> Orientador
- Inicia con 4 -> Profesor

IdMensajes se generan aleatoriamente con rand()

>>>>>>>>>>>> NOTA 1
Para controlar el inicio de sesión es importante que en cada archivo
lo primero que este sea la siguiente fracción de código:

<?php
session_start();
if(isset($_SESSION['loggedin']) && $_SESSION['loggedin'] == true)
{

}
else 
{
	header("Location: index.html");
	exit;
}
$now = time();
if($now > $_SESSION['expire'])
{
	session_destroy();
	header("Location: index.html");	
}
?>
